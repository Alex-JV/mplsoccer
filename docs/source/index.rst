mplsoccer
=========
mplsoccer is a Python plotting library for drawing soccer / football pitches in Matplotlib
and loading StatsBomb open-data.

.. image:: https://github.com/andrewRowlinson/mplsoccer/raw/master/docs/figures/README_animation_example.gif?raw=true
  :alt: Soccer pitch animation

An example of animation of `Metrica Sports <https://github.com/metrica-sports/sample-data`
tracking data using mplsoccer.
